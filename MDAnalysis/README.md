# Dockerfile to build a container for MDAnalysis

**Build from Dockerfile**

    docker build -t mdanalysis .
    docker images
    docker run -it --name=MDAnalysis-container mdanalysis

**Pull from Docker Hub**

    docker pull anjibabu480/mdanalysis
    docker run -it --name=MDAnalysis-container mdanalysis
